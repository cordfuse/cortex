# Custom Verbs

Define your own Cortex session commands here. Call them with a `/` prefix — e.g. `/weekly`.

The `/` prefix ensures custom verbs never conflict with built-in Cortex verbs (`hello`, `goodbye`, `status`, `sync`, `search`, `list verbs`).

Cortex loads this file at every `hello`. `list verbs` includes your custom verbs alongside the built-ins.

---

## Format

```
## /verbname
What this verb does — plain description. Be specific: what to read, what to ask, what to file.
```
