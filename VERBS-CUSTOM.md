# Custom Verbs — Personal

Add personal verbs here, or override framework verbs from `VERBS.md` by using the same name.

**The scribe manages this file.** To add a verb, describe what you want — the scribe writes it and commits. To override a framework verb, ask the scribe to redefine it here.

Invoked by **natural language** — say what you want, the scribe routes the intent. No slash prefix. Override example: define `## daily log` here to replace the framework version entirely. Verb names must not collide with built-in verb names (`hello`, `goodbye`, `status`, `sync`, `search`, `list verbs`, `list personalities`, `list actors`).

Framework verbs are activated/deactivated in `VERBS.md`. New verbs and overrides go here. Never mix them.
